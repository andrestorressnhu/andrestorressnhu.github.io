Enhancement 1: Weight Tracker Mobile App

Description: The artifact in this section is a mobile app to track a person’s weight. This app keeps track of weight over time. The app makes it convenient for the end user to see their progress towards a weight goal. Android Studio was the integrated development environment (IDE) utilized to create this app. JAVA programming language was used to create the functionality of the app. The Weight Tracker app was run and tested using a medium phone emulator on Android Studio IDE.

Enhancements:

  • Improved UI layout to be readable and visually appealing
  • Resolved issues with glitch not allowing users to input weight data
  • Enhanced data entry and added calendar input function

Technology Used: Java, Android Studio, XML, SQLite

Enhancement 2: Top 5 Destinations App

Description: The artifact is a Java-based desktop application called Top Five Destination List from CS 250. This application creates graphical user interfaces (GUI) in Java using Swing GUI imports. It presents a list of five travel destinations, each with an image and a brief description. Users can explore these destinations by navigating through them using page controls, which enable them to switch between different pages of content.

Enhancements:

  • Improved visual appeal and layout structure using Bootstrap and Dash components
  • Added more MongoDB query filters for advanced search functionality
  • Created a user-friendly interface for data visualization using hyperlinks

Technology Used: Python, Dash, MongoDB, Pandas, HTML, CSS

Enhancement 3: Grazioso Salvare Dashboard

Description: The selected artifact is a data visualization dashboard developed using Python, Dash, and MongoDB. This project was initially created during my CS 340 Client/Server Development. The dashboard is designed to connect to a live animal shelter database, allowing users to filter, view, and analyze rescue animal data through an interactive web interface. The database was the Austin Animal Center (AAC). I included this project in my portfolio because it showcases real-world software development skills, specifically in full-stack application design. This artifact shows my ability to integrate a non-relational database like MongoDB with a Python-based front end to create a real-time filtering and querying functionality based on user input.

Enhancements:

  • Optimized overall structure of the Dashboard and improved layout responsiveness
  • Optimized data fetching and rendering for better performance
  • Added page navigation, numbering, and enhanced accessibility

Technology Used: Python, Dash, MongoDB, Pandas, HTML, CSS
